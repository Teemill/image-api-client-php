## Installation

You can install the package via composer:

```bash
composer require teemill/image-api-client-php
```

## Usage

```php
$skeleton = new Teemill/ImageApi\Client();
echo $skeleton->echoPhrase('Hello, Teemill/ImageApi!');
```
